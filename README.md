# Voz Ativa 🎤

O **Voz Ativa** é um assistente de transcrição em tempo real focado em acessibilidade pedagógica. Ele permite que alunos capturem falas de professores e palestrantes, organizando-as em um ambiente digital limpo e funcional.

## ✨ Funcionalidades
- **Transcrição em Tempo Real**: Captura de áudio para texto com feedback visual imediato.
- **Categorização**: Filtros para aulas, saúde ou conversas diversas.
- **Geração de PDF**: Exportação do conteúdo transcrito com um clique para facilitar revisões.
- **Ficha Cadastral Integrada**: Sistema de cadastro de perfil acadêmico enviado via Formspree.
- **Interface Minimalista**: Paleta de cores selecionada para reduzir o cansaço visual (Bege, Verde Oliva e Gold).

## 🛠️ Tecnologias Utilizadas
- **React.js**: Biblioteca principal do frontend.
- **Web Speech API**: Para reconhecimento de voz nativo do navegador.
- **jsPDF**: Biblioteca para geração de documentos PDF.
- **Formspree**: Serviço para gerenciamento de formulários e envio de e-mails.
